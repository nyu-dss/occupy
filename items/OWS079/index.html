Object Type
Signs
Description
Sign that reads in bold letters "Mic Check." The mic-check was a key part of the "people's mic" process where the crowd would repeat a speaker's words so that it was audible to everyone in the audience. Typically, someone yelled "mic-check" in order to get everyone's attention before a meeting.
